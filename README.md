## 🚀 Overview

Professional software engineering portfolio showcasing high-fidelity full-stack web applications, academic audit tools, and modern UI/UX design systems. This project is built with **React**, **TypeScript**, and **Tailwind CSS v4**, focusing on performance, accessibility, and premium aesthetics.

## 🛠️ Tech Stack

- **Core:** React 19, TypeScript
- **Styling:** Tailwind CSS v4, Framer Motion (for cinematic animations)
- **Data Visualization:** Recharts (for GitHub statistics tracking)
- **Forms:** React Hook Form
- **Icons:** Lucide React
- **Dev Tools:** Vite, ESLint

## ✨ Key Features

- **Dynamic Data-Driven Architecture:** The entire portfolio scales through a central data configuration.
- **Micro-Animations:** Fluid transitions and interactive bento cards using Framer Motion.
- **GitHub Stats Integration:** Visualized GitHub contribution history and language distributions.
- **Bilingual Interface Support:** Ready for English and Amharic content delivery.
- **SEO Optimized:** Full meta-tag suite and JSON-LD structured data for professional indexing.
- **Responsive & Dark Mode:** Mobile-first design with a seamless Alabaster Light and Cosmic Dark theme system.

## 🏁 Getting Started

### Prerequisites

- Node.js (v18+)
- npm / yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/meron-s/meron-shambelu-portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment:**
   Create a `.env.local` file and add your Gemini API key:
   ```env
   VITE_GEMINI_API_KEY=your_api_key_here
   ```

4. **Run development server:**
   ```bash
   npm run dev
   ```
